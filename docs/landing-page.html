Common Use Cases

Feedback-Driven Development for Modern AI

Automating Code Reviews

Capture PR feedback and feed it directly into the agent's memory to ensure future refactors follow team-specific style guides without manual instruction.

Self-Healing Multi-Agent Systems

When an agent fails a task, the feedback loop generates a "prevention rule" that other agents in the system can use to avoid the same bottleneck.

DPO for Domain-Specific LLMs

Collect high-quality human preference data on your company's proprietary codebase to fine-tune smaller, cheaper models to perform like GPT-4 on your stack.

Quick start

From zero to feedback in one command

bash — your machine
# 1. Scaffold local config (no account needed)
$ npx rlhf-feedback-loop init
Created .rlhf/
Created .rlhf/config.json
Created .rlhf/capture-feedback.js
 
# 2. Capture feedback from your agent
$ node .rlhf/capture-feedback.js \
    --feedback=up \
    --context="refactored auth middleware" \
    --what-worked="extracted token validation into pure fn"
Feedback captured — id: fb_1772663401_up
 
# 3. Or hit the Cloud API with your key
$ curl -X POST https://rlhf-api.up.railway.app/v1/feedback \
    -H "Authorization: Bearer rlhf_YOUR_KEY" \
    -H "Content-Type: application/json" \
    -d '{"feedback":"up","context":"auth refactor","tags":["auth"]}'
{
  "id": "fb_1772663401_up",
  "status": "captured",
  "version": "0.5.0"
}
